# XroneTech - Agricultural Drone Spraying Services

Smart agricultural drone spraying solutions that help farmers increase productivity and reduce costs. Professional drone services for modern farming.

## Project info

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start development server:
   ```sh
   npm run dev
   ```

4. Build for production:
   ```sh
   npm run build
   ```
